Chat Searcher

Search for a word or phrase in a player's chat, or leave Steam ID empty to see a global word leaderboard. Steam ID accepts SteamID64, profile URL, vanity URL, or vanity name.

Multi-party Log Searcher

Enter Steam ID(s) in any format (profile URL, vanity name, or SteamID64), one per line or separated by space/comma.

Frequent Co-players

See who you play with and against most often across logs we have. Enter Steam ID (profile URL, vanity name, or SteamID64). Optionally filter by gamemode and map name substring.

Player search by name

Find Steam accounts by a substring of the in-game name used in chat we have indexed. Names link to Steam profiles; rows are sorted by how many different logs include that name. Use at least three characters for fast search.
